开始：组件结构

首先写一下 HTML 结构。

我们用一个div.stars-container包裹所有的子组件。这个星空组件由三个部分组成：

  • stars-box：包含 3 组星星，每组星星的密度，位置，速度各不相同，以达到 3D 的效果。
    • stars
    • stars2
    • stars3
  • fog：雾气效果，由一个雾气材质贴图提供。
  • horizon：发光的地平线效果，由 linear-gradient 渐变实现。
